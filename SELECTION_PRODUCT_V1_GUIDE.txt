选品 Product V1.0 操作说明

一、版本说明
- 当前版本名称：选品 Product V1.0
- 当前能力：基础产品数据加载、类目浏览、搜索、排序、筛选、查看原链接
- 数据来源：1688 选品抓取结果，已生成到 products.json

二、文件说明
请保持 demo 文件夹完整，不要只单独移动某一个文件。
主要文件包括：
- index.html：首页类目页
- products.html：商品列表页
- products.json：商品数据
- start_preview.bat：Windows 启动脚本
- start_preview.command：Mac 启动脚本

三、Windows 使用方法
1. 双击 start_preview.bat
2. 会弹出一个终端窗口，请不要关闭这个终端窗口
3. 稍等 2 到 5 秒，浏览器会自动打开网页
4. 如果浏览器没有自动打开，请手动打开以下地址：
http://127.0.0.1:8123/index.html
5. 使用完成后，可以关闭终端窗口，网页本地服务也会随之停止

四、Mac 使用方法
1. 双击 start_preview.command
2. 如果系统拦截，请右键文件，选择‘用终端打开’
3. 终端启动后，请保持终端窗口不要关闭
4. 如果浏览器没有自动打开，请手动打开以下地址：
http://127.0.0.1:8123/index.html

五、手动启动方式
如果脚本无法正常运行，可手动启动本地服务。

Windows：
python -m http.server 8123

Mac：
python3 -m http.server 8123

然后手动打开：
http://127.0.0.1:8123/index.html

六、网页内可做的操作
- 首页点击类目进入商品列表
- 搜索商品名或供应商
- 按上架时间排序
- 按销量排序
- 按价格排序
- 按发货地筛选
- 按价格区间筛选
- 点击‘查看原链接’跳转到 1688 商品页

七、注意事项
- 必须保持 demo 文件夹完整
- 启动后不要关闭终端窗口，否则网页会断开
- 当前图片使用网络图片地址，所以演示电脑需要联网
- 第一次打开如果显示旧数据，请刷新浏览器
- 如果浏览器打不开，请直接复制地址到浏览器：
http://127.0.0.1:8123/index.html

八、常见问题
1. 双击后网页没打开
处理方法：手动在浏览器输入 http://127.0.0.1:8123/index.html

2. 网页打不开
处理方法：
- 检查终端窗口是否已关闭
- 检查电脑是否安装 Python / Python3
- 检查 8123 端口是否被占用

3. 图片没显示
处理方法：
- 检查网络是否正常
- 当前版本图片为在线图片地址，断网时可能无法显示

4. Mac 提示无法验证脚本
处理方法：
- 右键 start_preview.command
- 选择‘用终端打开’
- 或手动执行 python3 -m http.server 8123

九、当前版本定位
当前版本主要用于：
- 演示选品网页结构
- 演示类目商品数据加载
- 演示基础搜索、排序、筛选能力

当前版本暂未覆盖：
- 自动定时更新
- OpenClaw 自动触发
- 飞书自动回传
- 增量分析和去重分析
