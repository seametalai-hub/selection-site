演示包使用说明

Windows：
1. 保持整个 demo 文件夹完整，不要只双击某个 html。
2. 电脑需要安装 Python。
3. 双击 start_preview.bat。
4. 浏览器会自动打开本地预览页面。

Mac：
1. 保持整个 demo 文件夹完整。
2. 电脑需要安装 Python 3。
3. 双击 start_preview.command。
4. 如果系统拦截，先右键用终端打开，或执行 chmod +x start_preview.command。
5. 浏览器会自动打开本地预览页面。

说明：
- 当前版本数据来自 products.json。
- 当前图片使用商品网络图片地址，演示电脑需要联网。
- 如果后续要做完全离线版，需要把图片下载到本地 images 目录。
