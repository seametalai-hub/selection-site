## 当前支持的字段

每个商品对象需要这些字段：

```js
{
  image: "https://example.com/product.jpg",
  title: "商品名",
  price: 49.9,
  shipFrom: "浙江宁波",
  sales90d: 1280,
  listedAt: "2026-03-06",
  originalUrl: "https://example.com/item/123"
}
```

## 排序

- `newest`：上新时间 晚到早
- `oldest`：上新时间 早到晚
- `sales`：90天销量 高到低

## 图片处理建议

图片不能只写本机路径，比如 `C:\\images\\1.jpg`，那样换电脑就失效。

可用方案：

1. 直接保存商品原站图片 URL，网页用远程地址显示
2. 抓取图片后保存到项目相对目录，再把整个项目一起发给别人
3. 上传到图床/CDN，再在网页里使用公开链接

后续接入真实爬虫时，优先建议：

- 如果目标站图片允许外链：直接用远程图片 URL
- 如果目标站图片有防盗链：下载到项目 `images/` 目录后统一分发
